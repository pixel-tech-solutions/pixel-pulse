# Changelog

All notable changes to Pixel Pulse will be documented in this file.

## [1.0.0] - 2026-02-16

### Added
- Initial release of Pixel Pulse
- Floating quote display window with transparency
- 10 quote categories: Motivational, Bible, Love, Tech, Business, Success, Leadership, Wisdom, Encouragement, Life
- Quote matching system (combine quotes from different categories)
- Customizable display settings (font, size, color, position)
- Fade animations with configurable timing
- Click-through mode
- Hover brightness effect
- System tray integration
- Auto-start with Windows option
- About dialog with company information
- Splash screen with branding
- Professional WiX installer
- SQLite database with 100,000+ quotes offline
- Settings persistence
- Right-click context menu

### Features
- **Quote Display:** Beautiful floating quotes in top-right corner (configurable position)
- **Quote Categories:** 10 comprehensive categories with extensive quote database
- **Quote Matching:** Combine quotes from different categories (e.g., Motivational + Bible)
- **Customization:** Full control over appearance and behavior
- **Offline:** Works completely offline with local database
- **Professional Branding:** Pixel Tech Solutions branding throughout

### Technical Details
- Built with .NET 8.0 and WPF
- SQLite database for quote storage
- Entity Framework Core for data access
- Professional WiX installer
- Windows 10/11 compatible
