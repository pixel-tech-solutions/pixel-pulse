# Pixel Pulse

**Inspiration at Your Fingertips**

A professional desktop application that displays inspirational quotes and Bible verses on your desktop.

## Features

- **Beautiful Quote Display** - Clean, modern interface
- **Multiple Quote Categories** - Motivational, Business, Bible, Love, and more
- **System Tray Integration** - Access quotes instantly from your system tray
- **Customizable Settings** - Font size, colors, display modes
- **Matched Quote Pairs** - Display complementary quotes together
- **Click-Through Window** - Quotes appear behind other applications
- **Self-Contained Database** - No internet required after installation

## Installation

1. Download `PixelPulse-Setup.exe`
2. Run the installer
3. Launch Pixel Pulse from Start Menu or Desktop

## System Requirements

- Windows 10 or 11
- 4GB RAM minimum
- 500MB free disk space

## Usage

### Basic Usage
- **Right-click** the system tray icon for options
- **Left-click** to show/hide the main window
- **Refresh Quote** - Get a new random quote
- **Settings** - Customize appearance and behavior

### Quote Categories
- Motivational
- Business & Leadership
- Technology
- Life & Wisdom
- Love & Relationships
- Bible Verses
- Success & Achievement
- Encouragement

### Display Modes
- **Single Quote** - Display one quote at a time
- **Matched Pairs** - Display two complementary quotes
- **Alternating** - Switch between quotes automatically

## Privacy

Pixel Pulse is completely self-contained:
- No internet connection required after installation
- No data collection or telemetry
- All quotes are stored locally
- No external dependencies
