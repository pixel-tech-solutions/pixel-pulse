PLACEHOLDER: Application Icon Required

This folder requires an icon.ico file.

See ICON_CREATION_GUIDE.md for detailed instructions on creating the icon.

Quick Steps:
1. Design a 256x256 pixel icon with brand colors (Purple #6B46C1, Blue #3B82F6)
2. Create multiple sizes: 16x16, 32x32, 48x48, 256x256
3. Convert to ICO format
4. Save as icon.ico in this folder

The icon will be used for:
- Application window title bar
- Taskbar icon
- System tray icon
- Start Menu icon
- Desktop shortcut icon
