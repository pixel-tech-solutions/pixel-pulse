# Installer Assets Required

This folder should contain the following installer graphics:

## Required Files

1. **banner.bmp** (493x58 pixels)
   - Top banner for installer dialogs
   - Should include "Pixel Tech Solutions" branding
   - Purple (#6B46C1) to Blue (#3B82F6) gradient background
   - Professional design with company name

2. **dialog.bmp** (493x312 pixels)
   - Background for installer dialogs
   - Subtle pattern or gradient
   - Brand colors (purple/blue theme)
   - Professional appearance

3. **icon.ico** (Multiple sizes: 16x16, 32x32, 48x48, 256x256)
   - Application icon
   - Pixel-themed design
   - Purple (#6B46C1) and Blue (#3B82F6) colors
   - Modern, professional appearance

## Design Guidelines

- Use brand colors: Purple (#6B46C1), Blue (#3B82F6), Gold (#F59E0B)
- Maintain professional appearance
- Ensure readability
- Consistent with application branding

## Creating the Assets

You can create these assets using:
- Adobe Photoshop/Illustrator
- GIMP (free alternative)
- Online icon generators
- Professional graphic design tools

The icon should match the application icon used in the main application.
