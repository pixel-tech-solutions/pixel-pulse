PLACEHOLDER: Installer Graphics Required

This folder requires the following graphics files:

1. banner.bmp (493x58 pixels, 24-bit BMP)
   - Installer banner with "Pixel Tech Solutions" branding
   - Purple to Blue gradient background

2. dialog.bmp (493x312 pixels, 24-bit BMP)
   - Subtle dialog background
   - Brand colors, non-distracting

3. icon.ico (Multiple sizes: 16, 32, 48, 256)
   - Can be same as application icon
   - Used in installer dialogs

See GRAPHICS_CREATION_GUIDE.md for detailed instructions.

IMPORTANT: Files must be in 24-bit BMP format for WiX compatibility.
ICO files must contain multiple embedded sizes.
